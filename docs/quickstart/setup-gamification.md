# Configurar gamificação
Neste tutorial vamos criar uma gamificação super simples onde os jogadores ganham pontos ao completar um desafio. O objetivo é conhecer as funcionalidades básicas para configurar e testar uma estratégia de gamificação.


## # [Funifier Studio](https://my.funifier.com/login)
Abra o navegador web de sua preferência e acesse o Funifier Studio
